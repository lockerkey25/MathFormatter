How to use

  • Type math directly — MathQuill formats as you type
  • Word shortcuts: type the word (e.g. sqrt, pi, int) and it auto-converts
  • Symbol shortcuts: -> → →, => → ⇒, <= → ≤, >= → ≥
  • Matrices: type pmatrix then Space → inserts 2×2 matrix; Tab / Shift+Tab to navigate cells
  • Accents: type vec, hat, bar etc. then the letter to accent
  • ^ for superscript, _ for subscript; Space to exit the box
  • Backspace immediately after an auto-conversion reverts it to the original typed text
